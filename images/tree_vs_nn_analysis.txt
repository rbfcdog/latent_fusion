
KEY FINDINGS:

1. EMBEDDING SPACE STRUCTURE
   • Concatenated embeddings (2560-dim) form sparse, separable patterns
   • Tree models excel at finding axis-aligned splits in high dimensions
   • Neural networks struggle with low-signal-to-noise in this space

2. TREE MODEL ADVANTAGES
   • Decision Tree: Near-perfect fit (R²=0.9942) → No interaction learning needed
   • Gradient Boosting: Sequential refinement (R²=0.9877) → Better generalization
   • Random Forest: Ensemble averaging (R²=0.9025) → Robust patterns
   • All achieve <1ms inference time

3. NEURAL NETWORK DISADVANTAGES
   • TemporalFusion (R²=0.3320): Architecture too complex for simple patterns
   • Attention mechanism collapsed (unused)
   • 3560 parameters underutilized (embeddings already processed)
   • MSE loss inadequate for this regime

4. CURSE OF DIMENSIONALITY PARADOX
   • Expected: NN easier in high-dim
   • Actual: Tree excels here (axis-aligned splits)
   • NN needs learned representation, not raw embeddings
